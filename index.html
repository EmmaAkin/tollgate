    A Python/Django-based Captive Portal System for LAN Parties.

    Introduction

    tollgate is a Python/Django-based captive portal system for Linux. It is primarily for LAN parties, but can run in other (primarily non-commercial) environments as well. It's licensed under the terms of the Affero GNU General Public License, version 3.

    tollgate has been under development since late 2008, and private releases were deployed since the start of 2009. It's run everything from 200-person events down to managing the internet connection for five people in a granny flat.

    The project was open sourced in 2011.

    The current stable release is 3.0.1 (Cavity Protection), released on 13th May, 2012.

    We're currently working on 3.0.2 which aims to fix some bugs in the software. This is undergoing stabilisation, and to be released soon.

    After this, we'll be working on 3.1.0 (Tartar Control), which will introduce new features to tollgate. This is slated to be released when we're happy with it.

    Getting Started
